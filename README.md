# Broadway Opticians — Design Directions

Four distinct web design directions for **Broadway Opticians** (The Mall Road, Mussoorie — est. 1960),
built on the boutique's real inventory. Each direction includes a landing page, a product page, and two
brand-themed pages (Gucci and Vysen) that re-skin into each brand's own visual identity.

Open **index.html** for the pitch hub linking all four:

1. **The Atelier** — optician's craft; trial-lens signature
2. **Refraction** — dark night-gallery; light refraction
3. **The Mall Road** — Mussoorie hill-station heritage
4. **New Wave** — porcelain fashion energy

Static HTML/CSS + vanilla JS. No build step.
